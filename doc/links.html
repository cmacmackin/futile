Contains a link derived types from which various linked-list type structures can be made. End users most likely wouldn't make direct use of anything in this module.

Date: November, 2014

Author: Christopher MacMackin

Description of Types

link

 
type, public :: link
    private
    class(*), pointer :: contents => null ()
The datum contained by the link. 
    class(link), pointer :: next => null ()
The next link in the chain. 
contains
    procedure :: get_contents
Returns the value held by the link. 
    procedure :: set_contents
Sets the value to be held by the link. 
    procedure :: get_next
Returns the next link in the chain. 
    procedure :: set_next
Sets the next link in the chain. 
end type link
The component element of a linked-list. The procedures here do not protect against memory leaks because this is not meant to be used as an opaque object.

double_link

 
type, public, extends(link) :: double_link

The previous link in the chain 
contains
    procedure :: get_prev
Returns the previous link in the chain. 
    procedure :: set_prev
Sets the previous link in the chain. 
end type double_link
The component object of a doubly-linked list. The procedures here do not protect against memory leaks because this is not meant to be used as an opaque object.
