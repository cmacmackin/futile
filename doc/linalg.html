Provides subroutines and functions to perform linear algebra tasks.

Author: Christopher MacMackin

Description of Subroutines and Functions

linsolve

 
public subroutine linsolve (matrix, reval, vals)
    real(kind=8), intent(in), dimension(:,:) :: matrix
A square, 2-dimensional array, which is the LU factorization of the matrix containing the coefficients of the linear system to be solved. 
    integer, intent(out) :: reval
    real(kind=8), intent(inout), dimension(size(matrix,1)) :: vals
end subroutine linsolve
Solves a system of linear equations using the (pre-computed) LU decomposition of the matrix describing the system. The decomposition can be computed using the subroutine lu_decomp(), below.

ls_fit

 
public subroutine ls_fit (coeffs, lsval, order, reval, xvals, yvals)
    real(kind=8), intent(out), dimension(:) :: coeffs
A 1-dimensional array with a length of at least 'order,' in which the coefficients of the fit will be returned. 
    real(kind=8), intent(out) :: lsval
The square difference values (error) of the fit. 
    integer, intent(in) :: order
The order of the polynomial to be fit to the data. 
    integer, intent(out) :: reval
The return code of the subroutine. A value of 0 indicates that the subroutine executed as expected. A value of -1 indicates that the matrix of coefficients passed to either linsolve() or lu_decomp() was not square, as it should have been. A value of -2 indicates that the coeffs array passed to ls_fit() is not large enough to hold all of the coefficients for the desired fit. A value of -3 means that there is insufficient data for ls_fit() to fit a polynomial of the desired order. 
    real(kind=8), intent(in), dimension(:) :: xvals
A 1-dimensional array containing the independent variable. 
    real(kind=8), intent(in), dimension(:) :: yvals
A 1-dimensional array containing the dependent variable. 
    ! Calls: linsolve, ludecomp
end subroutine ls_fit
Finds the least-square polynomial fit for a set of data.

lu_decomp

 
public subroutine lu_decomp (matrix, reval)
    real(kind=8), intent(inout), dimension(:,:) :: matrix
A square, 2-dimensional array, which is to be factored. The factorization is returned within this matrix as well. The diagonal of ones in the upper triangular matrix is not explicitly represented, as the values of these elements are known. 
    integer, intent(out) :: reval
The return value of the subroutine. A value of 0 indicates that the subroutine executed as expected. A value of -1 indicates that the matrix of coefficients passed to either linsolve() or lu_decomp() was not square, as it should have been. A value of -2 indicates that the coeffs array passed to ls_fit() is not large enough to hold all of the coefficients for the desired fit. A value of -3 means that there is insufficient data for ls_fit() to fit a polynomial of the desired order. 
end subroutine lu_decomp
Performs LU decomposition on the provided square matrix.
